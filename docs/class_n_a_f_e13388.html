Detailed Description

NXP Analog Front End class library for Arduino

Author
Tedd OKANO
Version
0.1.1
Date
27-June-2023

Copyright: 2023 Tedd OKANO Released under the MIT license

A simple class library for NXP Analog Front End: NAFE13388

Example:

#include <NAFE13388.h>
NAFE13388 afe;
void setup() {
Serial.begin(9600);
while (!Serial)
;
SPI.begin();
afe.begin();
Serial.println("\n***** Hello, NAFE13388! *****");
afe.logical_ch_config(0, 0x1150, 0x00AC, 0x1400, 0x0000);
afe.logical_ch_config(1, 0x3350, 0x00A4, 0x1400, 0x3060);
Serial.println("logical channel 0 and 1 are shown in micro-volt");
}
void loop() {
Serial.print(afe.read(0));
Serial.print(", ");
Serial.println(afe.read(1));
delay(100);
}

void NAFE13388::begin ( void  )

Begin the device operation

NAFE13388 initialization. It does following steps (1) Set pins 2 and 3 are input for nINT and nDRDY (2) Set pins 5 and 6 are output and fixed to HIGH for ADC_SYN and ADC_nRESET (3) Call reset() (4) Call boot()
